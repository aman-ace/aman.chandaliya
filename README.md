# Aman Chandaliya — Personal Research Site

Personal website showcasing research, writing, and policy analysis on public finance, state budgeting, and infrastructure resilience.

## Structure

- `index.html` — Homepage with intro and featured research
- `about.html` — Background, focus areas, and methods
- `research.html` — Index of research projects
- `articles.html` — Short-form articles and commentary
- `contact.html` — Contact and links
- `research/` — Individual research paper pages
- `assets/` — Images, PDFs, icons
- `style.css` — Site styles
- `_config.yml` — Jekyll/GitHub Pages config

## Local Preview

Open `index.html` directly in a browser, or serve with:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.

## Hosting

Deployed via GitHub Pages from the `main` branch.
